Jun 3, 2019 | Hexo

以前博客在CSDN，自己用Adblock+Tampermonkey插件弄了一个整洁版的，但还是放弃了，理由是：广告真的太多，而且CSDN内相同的博文太多了。
期间想过自己搭建一个博客，但是由于强迫症，考虑到页面的美观和数据库设计问题以及日后迁移问题，所以迟迟没有动手。最后还是决定把数据托管在GitHub，并使用Hexo。
